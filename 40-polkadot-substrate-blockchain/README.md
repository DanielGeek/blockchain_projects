# Substrate node template and Substrate front-end template
# Commands to use
- Run backend node
  - `cd substrate-node-template`
  - `./target/release/node-template --dev`

- Run Frontend
  - `cd substrate-front-end-template`
  - `yarn install`
  - `yarn start`
